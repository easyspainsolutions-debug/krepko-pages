Política de privacidad / Политика конфиденциальности

Действует с 2026-04-29 · Última actualización: 29 abril 2026
Краткая суть. Krepko собирает минимум данных (имя, контакт, тема обращения) только чтобы связаться с вами по вашему запросу. Не передаёт третьим лицам, кроме партнёров по конкретной услуге (нотариус, банк, gestor). Удаляет по запросу. Не отправляет спам.

2. Какие данные собирает Krepko (Datos personales que tratamos)

Через сайт krepko.es, Telegram-бот @krepko_es_admin_bot и WhatsApp +34 641 048 296 Krepko может собирать:

  • Имя (как обращаться)
  • Контакт — Telegram username, номер WhatsApp или email
  • Город в Испании
  • Тема обращения — нотариус, банк, ВНЖ, документы и т.д.
  • Краткое описание ситуации, которое вы добровольно сообщаете
  • Технические данные при заходе на сайт — Google Analytics 4 (анонимизированный IP, источник трафика, поведение на странице) — для аналитики посещаемости

Krepko не запрашивает и не хранит паспортные данные, NIE, номера счетов, фотографии документов или любые иные чувствительные данные через сайт/бот. Эти данные передаются только при заключении конкретного договора на услугу — отдельным каналом, с подписанием соответствующего соглашения.

3. Цели обработки (Finalidad)

  1. Связаться с вами в ответ на ваш запрос (звонок, сообщение, консультация)
  2. Подготовить расчёт стоимости и план действий по вашему запросу
  3. Заключить договор на услугу, если вы согласны
  4. Аналитика посещаемости сайта (Google Analytics 4) — для улучшения контента

Krepko не использует данные для маркетинговой рассылки без вашего отдельного согласия. Не продаёт и не передаёт третьим лицам в рекламных целях.

4. Правовое основание (Base legal)

  • Согласие (Art. 6.1.a RGPD) — для контактных данных и сообщений, которые вы добровольно отправляете через бот, форму или WhatsApp
  • Исполнение договора (Art. 6.1.b RGPD) — для данных, нужных для оказания услуги после её заказа
  • Законные интересы (Art. 6.1.f RGPD) — для базовой аналитики посещаемости (агрегированные, неперсональные данные)

5. Получатели данных (Destinatarios)

Ваши данные могут передаваться:

  • Партнёры по услуге — нотариус, банк, миграционный юрист, gestor — только в объёме, нужном для исполнения вашего запроса, и только с вашего согласия
  • Поставщики технических услуг — Google (Analytics, Sheets), Telegram, WhatsApp Business, Cloudflare. Они выступают как encargados del tratamiento (data processors) и связаны соответствующими соглашениями
  • Государственные органы — при наличии законного запроса (суд, налоговая)

6. Международная передача данных (Transferencia internacional)

Часть технических поставщиков (Google, Telegram) могут хранить данные в США. Эти передачи регулируются:

  • EU-US Data Privacy Framework для Google (сертификация Privacy Shield 2.0)
  • Стандартные договорные положения (Standard Contractual Clauses, SCC) для остальных

7. Срок хранения (Plazo de conservación)

  • Контактные данные без последующей сделки: 12 месяцев с момента последнего контакта, затем удаление
  • Данные клиентов с заключённым договором: срок исковой давности по испанскому законодательству (обычно 5-10 лет) для документации
  • Аналитика GA4: 14 месяцев (стандартное удержание Google)

8. Ваши права (Derechos ARSOPL)

Согласно RGPD и LOPDGDD у вас есть права:

  • Acceso (доступ) — узнать какие данные о вас хранит Krepko
  • Rectificación (исправление) — попросить исправить неточные данные
  • Supresión (удаление, «право на забвение») — попросить удалить ваши данные
  • Oposición (возражение) — возразить против обработки данных по легитимным интересам
  • Portabilidad (переносимость) — получить ваши данные в машиночитаемом формате
  • Limitación (ограничение) — попросить ограничить обработку в спорных случаях
  • Retirada de consentimiento — отозвать согласие в любой момент (на будущую обработку)

Чтобы воспользоваться правами — напишите на easyspainsolutions@gmail.com или в WhatsApp с пометкой «Solicitud RGPD». Krepko ответит в течение 30 дней.

Если вы считаете что ваши права нарушены — вы можете подать жалобу в Agencia Española de Protección de Datos (AEPD): www.aepd.es

9. Cookies

Сайт использует Google Analytics 4 cookies для агрегированной аналитики посещаемости. Никаких рекламных или трекинговых cookies. Вы можете отключить cookies в настройках браузера — сайт продолжит работать.

10. Безопасность данных

  • HTTPS на всём сайте (TLS 1.3, GitHub Pages)
  • Двухфакторная аутентификация на всех аккаунтах (Google, Railway, Telegram)
  • Минимизация данных — Krepko собирает только то, что нужно для услуги
  • Резервное копирование с шифрованием
  • Логи доступа — нет публичных endpoints с прямым доступом к личным данным

11. Изменения политики

Krepko может обновлять эту политику. Существенные изменения будут обозначены явно с новой датой. История изменений хранится в git репозитории сайта (открытый код).
